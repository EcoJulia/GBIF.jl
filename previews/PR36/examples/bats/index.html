Rank-abundance curves of bats in Europe

In this example, we will use the GBIF package to produce a rank-abundance curve of Chiroptera species in Europe, based on data from 2000 to 2005.

using GBIF
using DataFrames
using Query
using StatsPlots

bats = GBIF.taxon("Chiroptera"; strict=false)
occ = occurrences(bats, "continent" => "EUROPE", "year" => (2000, 2005), "limit" => 300)
while length(occ) < size(occ)
occurrences!(occ)
end
